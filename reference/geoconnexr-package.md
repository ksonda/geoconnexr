# geoconnexr: Geoconnex discovery and watershed data snapshots

`geoconnexr` provides typed, versioned foundations for discovering and
packaging water data described through the Geoconnex ecosystem. The
package is currently experimental and its network-facing APIs are being
developed behind fixture-backed contracts.
