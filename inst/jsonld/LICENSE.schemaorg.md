# Schema.org context notice

`schemaorg-context-v30.0.json` is a snapshot of the official Schema.org
JSON-LD context published by the Schema.org sponsors. It was obtained from
<https://schema.org/docs/jsonldcontext.json> on 2026-07-13 while the site
reported release V30.0. The bundled file differs from the retrieved byte stream
only by one terminal LF.
