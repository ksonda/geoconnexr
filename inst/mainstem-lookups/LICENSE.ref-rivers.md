# ref_rivers lookup data license

The full optional `ref_rivers` lookup installed by geoconnexr is not bundled
with this package. A hand-minimized six-row excerpt is bundled only as a test
fixture. Both are derived from work that the upstream repository dedicates to
the public domain under the [CC0 1.0 Universal dedication](https://github.com/internetofwater/ref_rivers/blob/fd42072b1e9bf9de33cd047052b99508753c7fea/LICENSE).
